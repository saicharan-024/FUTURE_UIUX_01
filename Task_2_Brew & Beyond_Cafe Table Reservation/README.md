# Task 2 - Mobile App UI for  Booking

# Business Chosen
 Brew & Beyond – Cafe Table Reservation App

## Project Overview

Brew & Beyond is a modern mobile application UI designed for cafe table reservations.

The app allows users to:

- Login securely
- Select tables
- Choose date & time slots
- Confirm reservations
- Complete payments

## Tools Used

- Figma
- Githib

## Features

- Modern Dark Theme
- Gold Accent Design
- Table Reservation System
- Booking Confirmation
- Payment Integration
