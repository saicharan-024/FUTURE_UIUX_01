# Task 1 - Local Service Business Website Redesign

## Business Chosen
Smile Care - Dental Clinic Website

## Objective
Smile Care is a modern dental clinic website UI focused on improving online appointment bookings and building trust with patients through a clean and conversion-focused design.

## Business Type:
Dental Clinic Website

## Target Audience:
Patients looking for dental services online

## Goal:
Increase appointment bookings and trust

## UX Decisions:
- Clear CTA buttons
- Simple navigation
- Trust-building testimonials
- Mobile responsive layout

## Conversion Improvements:
- Easy booking flow
- Strong visual hierarchy
- Clean spacing

## Pages Designed
- Homepage
- Service Detail Page
- Contact / Lead Generation Page
- Mobile Responsive Screens

## Screens

1. Task-1
2. Splash Screen
3. Login 
4. Choose Service
5. Dental Services-1
6. Dental Services-2
7. Select Date & Time
8. Booking Confirmation
 

## Tools Used
- Figma


## Features
- Clear CTA sections
- Modern UI
- Mobile responsive layout
- Testimonials & trust elements
- Appointment focused UX
